# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-04-16

### Added
- Initial release
- ASCII branch visualization with proper branching/merging lines
- Commit details: hash, author, date, message
- Keyboard navigation (vim keys, arrows, page up/down)
- Mouse support (click to select, scroll)
- Search and filter by author/date
- Color-coded branches
- Repository auto-discovery
- Pure Python git parser (no git CLI dependency)
- Diff viewer
- Textual-based TUI
