## Quick Start

Each project has its own setup instructions. Check individual `README.md` files.

```bash
# Example: Python projects
cd <project-name>
pip install -r requirements.txt
python app.py

# Example: Go projects
cd <project-name>
go build -o app .
./app

# Example: Rust projects
cd <project-name>
cargo build --release
./target/release/<project>
```
